# 이스트 소프트 오르미 백엔드

##  과정 소개
이스트소프트 오르미 백엔드 과정은 백엔드 개발자로 성장하기 위한 핵심 기술들을 학습하는 과정입니다.  
Java, Spring(SQL, ERD 등), TDD 실무 중심 커리큘럼을 통해  백엔드 개발자로서 필요한 역량을 체계적으로 다집니다.

---

##  학습 내용
- Java 기초 및 객체지향 프로그래밍
- Spring 프레임워크
- JPA, JDBC 등 ORM 활용
- Git & GitHub 협업 실습
- 배포 및 AWS 클라우드 기초
- 팀 프로젝트 및 실무형 미니 프로젝트
